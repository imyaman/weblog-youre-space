GCP UPDATES | FEBRUARY 4, 2019
COMPUTE
NVIDIA® Tesla® T4 on Compute Engine: beta
NVIDIA® Tesla® T4 with new NVIDIA ray-tracing features is now available for use with your machine learning inference, ML training, and graphics-intensive visualization workloads. Google Cloud Platform offers these GPUs – which feature NVIDIA Tensor Cores built for ML and 16 GB of GPU memory – for as low as $0.29 USD per hour on preemptible VM instances. Documentation
Support for Go 1.11 on Cloud Functions: beta
Cloud Functions now supports the latest version of Go and Go language features, letting you write functions that scale in response to load and integrate seamlessly with Google Cloud events. Blog
DATABASES
Cloud Spanner now available in the Sydney region (australia-southeast1): GA
You can now provide lower-latency reads and writes for your users and services located in the Sydney region. Documentation
Cloud Spanner now available in the London region (europe-west2): GA
For any regional configuration, Cloud Spanner maintains three read-write replicas, each within a different GCP zone in that region. Documentation
NETWORKING
Internal Load Balancing support for all ports: beta
This release removes the constraints for defining forwarding rules, allowing you to match all ports. This provides you with more flexibility when running multiple applications over TCP. Documentation
API MANAGEMENT
Monetization in the new Apigee Edge experience: GA
The API monetization feature is now available in the new Apigee Edge experience, giving you flexible ways to generate revenue from your APIs. Previously, this feature was only available in the classic experience. Documentation
MIGRATION
Transfer Appliance now available in Canada: GA
You can use Transfer Appliance to securely migrate large volumes of data – from hundreds of terabytes to a petabyte – to GCP without disrupting business operations. Contact your Google sales representative to learn more. Documentation
