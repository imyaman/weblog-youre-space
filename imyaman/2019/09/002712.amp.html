Learn more about new beta Recommender tools to be implemented in Google Cloud Platform.
MY CONSOLE

Hello Google Cloud Platform Customer,

We are writing to let you know that various Recommenders for Google Cloud Platform (GCP) are coming soon to beta. These features, which you will be able to opt out of before they are launched in the coming weeks, will provide recommendations about how to help make your cloud deployments more secure, cost effective and performant.

What do I need to know?

Recommenders automatically analyze usage patterns to help determine if resources and policies within Google Cloud are optimally configured. Data usage information is not shared in the process.

With the Cloud Identity and Access Management (Cloud IAM) Recommender, you can automatically detect overly permissive access policies and adjust them based on the access patterns of similar users in your organization. This is a follow up to our Alpha launch at Cloud Next this past April.

For example, if a set of permissions hasn’t been used in 90 days, the Cloud IAM Recommender may recommend that you apply a less permissive role. In this case, Cloud IAM recommendations are generated by analyzing the Cloud IAM permissions over a 90 day period for each customer individually. They serve to create an overall model that is used to recommend more secure Cloud IAM policies. Unlike rule-based recommendations, these models and recommendations improve over time.

The Google Compute Engine Rightsizing Recommender helps you choose the optimal virtual machine size for your workload. Our experience working with customers shows that they often initially provision machines that are too small or too large.

We will be releasing more Recommenders in the future.

What do I need to do?

You have multiple options. You can:

  • Do nothing to take advantage of these features at launch,
  • Sign up for a pre-briefing to learn more about these features before launch,
  • Opt-out prior to launch by filling out this form by September 19, 2019, or
  • Opt-out after launch by going to the Recommendation section in the Security & Privacy navigation panel from the Cloud Console, which will launch at the same time as the Recommenders.
Need more assistance?

Detailed GCP Recommenders documentation will be available at launch. If you have any questions or require assistance, please reply to this email to contact Google Cloud Support.

Thank you for being a valued Google Cloud Platform customer.
