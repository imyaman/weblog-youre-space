Dear Sangyong,

We are excited to announce that Splunk Light 6.6 is now available for download !

Splunk Light 6.6 makes it easier than ever for a wide range of users to leverage datasets, get answers and share insights.

In Splunk Light 6.6:
*Dataset explorer allows you to use a prepared dataset to create/schedule a report and export it in CSV format
*Dashboard drilldown makes it easier to build a guided dashboard navigation experience without writing any Simple XML

* Trellis layout enables you to build multiple visualizations powered by a single search to compare different segments of a dataset

Download the latest version today !
