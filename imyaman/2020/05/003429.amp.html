PRODUCT UPDATES | MARCH 2, 2020
DATA & ANALYTICS
Dataflow – observability: GA
View your pipeline, and any other job, with the Dataflow web-based monitoring interface. See and interact with a list of your Dataflow jobs. Check a job’s status, execution, and SDK version. And find links to information about the services running your pipeline. Blog | Documentation
COMPUTE
Compute Engine – sole-tenant VM tenancy migration: GA
Move between multi-tenant and sole-tenant nodes, or between sole-tenant node groups to adjust tenancy based on workload requirements. Blog | Documentation
Compute Engine – bring you own license (BYOL) V2: beta
Run your Windows workloads – and use existing Windows OS licenses – on Compute Engine. New features include resource pooling of sole-tenant nodes and the ability to live migrate BYOL instances for maintenance events to other nodes in the pool without any downtime. Documentation
Compute Engine – sole-tenant autoscaler: beta
Automatically manage the size of your sole-tenant node groups. Set up the node group autoscaler to create additional VM instances when there’s insufficient capacity and delete empty nodes to reduce costs. Documentation
Compute Engine – stateful managed instance groups: beta
Build highly available deployments of stateful workloads like databases, data-processing applications, legacy monolithic applications, and long-running batch computations with checkpointing. Improve uptime and resilience while preserving the unique state of each instance. Documentation
Compute Engine – managed instance group rollout verification: GA
Verify whether an update rollout has been completed with the gcloud command-line tool or the API. Or use the Google Cloud Console UI to see the current and planned number of instances being updated. Documentation
Compute Engine – configure regional managed instance group rebalancing: GA
Control whether to keep or disable redistribution behavior in regional managed instance groups – whether the regional managed instance group rebalances your instances proactively, as it does now by default, or opportunistically, when the group size changes. Documentation
CONTAINERS
Google Kubernetes Engine (GKE) – GKE Sandbox: GA
Keep your system secure and robust, simplify how you manage workloads, and optimize resource utilization in your cluster. GKE Sandbox allows you to deploy trusted workloads and sandboxed, untrusted workloads on the same node. Documentation
DEVELOPER TOOLS
Cloud Code – client library browser: GA
Access a consolidated list of your Google Cloud services coupled with their corresponding client libraries and documentation. Browse and enable Cloud APIs and add Cloud Client Libraries to your project from your Visual Studio Code or IntelliJ IDEs. Documentation
NETWORKING
Network Intelligence Center – Connectivity Tests: beta
Verify connectivity – and troubleshoot connectivity issues due to misconfiguration – in your Virtual Private Cloud (VPC) network with Connectivity Tests, a static configuration analyzer and diagnostics tool. Documentation
SECURITY & IDENTITY
Cloud Asset Inventory – real-time notification service: GA
Create subscriptions for continuous, real-time updates on resource and policy changes to monitor a range of supported resource types – as well as Cloud Identity and Access Management policies within an organization, folder, project, or specific resource for Google Cloud Platform and Anthos. Documentation
VPC Service Controls – new service integrations: GA
Explore the full range of supported products and services, which now include Cloud TPU, Cloud Vision, and Cloud Trace. Documentation
