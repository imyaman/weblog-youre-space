PRODUCT UPDATES  |  MAY 14, 2020

 

 

AI & MACHINE LEARNING

Video Intelligence API – logo recognition: GA

Automatically detect, track, and recognize the presence of over 100,000 brands and logos in video content.

Documentation

 

COMPUTE

Recommender – insight types: beta

Configure Recommender to provide detailed insights, recommendations, and automations using insight types, including firewall and IAM types, to run your Google Cloud environment in a more cost-efficient, secure, and less labor-intensive way.

Documentation | Product page

 

Compute Engine – AMD EPYC-based N2D VM instances: GA

Use N2D virtual machine (VM) instances for both general-purpose workloads, like web servers and testing, and workloads that require high memory bandwidth, like crash analysis, financial modeling, and rendering. Available in large machine types with up to an industry-leading 224 vCPUs.

Blog

 

Compute Engine – OS patch management: GA

Opt in to enable automatic patching for the guest environment on your Compute Engine VM instances with updates released by OS providers.

Blog | Documentation

 

Compute Engine – Shielded VM now default: GA

Defend against rootkits and bootkits, detect low-level platform compromises of your VM instances, and reduce the risk of exfiltrated data with Shielded VM – now the default for all Compute Engine users – at no additional charge.

Blog | Product page

 

CONTAINERS

Google Kubernetes Engine (GKE) – Microsoft Windows Server containers: GA

Take advantage of the agility, speed of deployment, and simplified management of your Windows Server apps with containers on GKE. Run Windows Server and Linux containers side by side in the same cluster for a central management plane for both platforms.

Blog | Documentation

 

HEALTHCARE & LIFE SCIENCES

Cloud Healthcare API: GA

Securely build clinical and analytics solutions – and ingest and manage key data using popular healthcare data standards such as HL7® FHIR®, HL7® v2, and DICOM® – with our fully managed, highly scalable, enterprise-grade development environment.

Blog | Documentation | Product page

 

Cloud Healthcare API – FHIR streaming to BigQuery: GA

Configure the Cloud Healthcare API to automatically stream new and updated FHIR resources to a target BigQuery dataset.

Documentation

 

Cloud Healthcare API – HL7 v2 schematized parser and notifications: GA

Specify a custom schema for parsing incoming HL7 v2 messages into JSON format while preserving grouping information. Configure HL7 v2 stores to publish notifications on multiple Pub/Sub topics and use a filter to manage the notifications sent to each topic.

Documentation | Documentation

 

Cloud Life Sciences – new regions: beta

Access Cloud Life Sciences in new regions, including asia-northeast1 (Tokyo), asia-southeast1 (Singapore), europe-west2 (London, UK), europe-west4 (Netherlands), northamerica-northeast1 (Montreal), us-west2 (Los Angeles), and the us multi-region (data centers in the US).

Documentation

 

INFRASTRUCTURE

New Google Cloud Platform region: Las Vegas

Get added capacity and the flexibility to distribute your workloads across the western United States – including our existing cloud regions in Los Angeles, Salt Lake City, and Oregon – with our newest region: Las Vegas (us-west4).

Blog﻿

 

NETWORKING

Network Intelligence Center – Firewall Insights: beta

Gain visibility into your firewall usage and detect configuration issues to better understand your firewall rules and simplify your settings. Examine metrics in the Monitoring API and your Cloud Console.

Documentation

 

SECURITY & IDENTITY

Security Command Center – Security Health Analytics: GA

Prevent threats with continuous monitoring of security best practices, detect security misconfigurations in your Google Cloud infrastructure, such as public buckets and open firewalls, and generate vulnerability reports in Security Command Center.

Documentation

 

Event Threat Detection: GA

Automatically scan for and quickly identify high-risk and costly threats in various types of logs in your Google Cloud environment, including malware, cryptomining, unauthorized access, outgoing DDoS attacks, and brute-force SSH.

Documentation

 

Cloud Asset Inventory – IAM Policy Analyzer: beta

Enable organization and folder administrators to better understand identity and access management policies and address audit- and compliance-related tasks. Perform access administration and determine which users have access to specific Google Cloud resources.

Documentation
