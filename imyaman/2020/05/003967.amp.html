Deletion notice for your Google Cloud Shell home directory

Hello Cloud Shell user,

It's been over 120 days since you opened Cloud Shell from the Google Cloud Platform console. In 7 days, your Cloud Shell home directory will be automatically scheduled for deletion.

To keep your Cloud Shell home directory and its data, just log in and open Cloud Shell.

Open Cloud Shell to retain or backup your data

If you do nothing, you'll still have access to Cloud Shell: just start a new session and you're good to go. Your Cloud Platform projects won't be deleted and other services like App Engine or Compute Engine won't be affected.

Questions? Visit the Cloud Shell help page.
