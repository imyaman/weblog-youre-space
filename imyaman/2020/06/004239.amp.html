PRODUCT UPDATES | JUNE 15, 2020
AI & MACHINE LEARNING
Video Intelligence API – face and person detection: beta
Detect faces and people with timestamps and bounding boxes, attributes like clothing and hair, and by 2D pose using 17 body landmarks. Documentation | Documentation
Media Translation – real-time audio translation: beta
Conduct real-time translation directly from streaming audio input from a microphone or from prerecorded audio files with low latency, enhanced accuracy, and support for a range of media types. Documentation
COMPUTE
Compute Engine – bring your own IP addresses (BYOIP): beta
Bring your public prefixes to Google Cloud and we’ll manage the route announcement for you. Access to BYOIP is subject to approval and, in order to be eligible, you must own and bring a /24 block of IPs or larger. Documentation
DATA ANALYTICS
BigQuery – dynamic SQL: beta
Execute dynamically generated SQL strings from within BigQuery using the EXECUTE IMMEDIATE statement. Blog | Documentation
DATABASES
Cloud Spanner – transaction statistics tables: GA
Investigate performance issues, check for slow-running transactions that might cause contention, and identify potential bottlenecks, such as large volumes of updates to a particular column. Documentation
DEVELOPER TOOLS
Cloud Functions – Java 11 support: beta
Write Cloud Functions using your favorite JVM languages, including Java, Kotlin, Groovy, and Scala with our Functions Framework for Java, as well as Spring Cloud Functions and Micronaut. Blog
NETWORKING
Cloud Load Balancing – external load balancer header- and parameter-based routing: GA
Route traffic based on host and path rules with external load balancer URL maps. Use a path matcher to match an HTTP(S) request path or path prefixes with backend services and define a host rule that sends traffic to a path matcher based on the host component. Documentation
SECURITY & IDENTITY
Cloud Identity and Access Management – manage Google groups in the Cloud Console: beta
Create and edit Google groups within the Cloud Console without the need to switch interfaces for group management when administering identity and access management permissions. Documentation
