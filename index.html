PRODUCT UPDATES | JANUARY 27, 2020
AI & MACHINE LEARNING
AI Platform – built-in image classification and object detection algorithms: beta
Train a model without writing code with built-in image classification and object detection algorithms. Use hyperparameter tuning for greater accuracy, use TPUs for fast training, and export models for prediction, either locally or deployed to AI Platform Prediction. Documentation | Documentation
Virtual Private Cloud (VPC) – use VPC Service Controls with AI Platform training: beta
Mitigate the risk of data exfiltration from your training jobs by running them from a project inside a service perimeter using VPC Service Controls to ensure your data doesn’t leave the perimeter – including training data your job accesses and artifacts it creates. Documentation
COMPUTE
Compute Engine – operations.wait: GA
Reduce the frequency of operation status polling, and help make more efficient use of your queries-per-second quotas. Global, regional, and zonal operations.wait methods wait for the specified operation resource to return as done, or for the request to hit a two-minute deadline, and then retrieve it. Documentation
DATA ANALYTICS
BigQuery ML – data preprocessing: GA
Streamline and simplify your machine learning model builds – and keep transformations consistent between training and inference. Preprocess and transform your data with simple SQL functions and automatically apply those transformations during the prediction phase. Blog | Documentation | Tutorial
NETWORKING
VPC – Private Google Access for on-premises hosts: beta
Access Google APIs and services privately from on-premises hosts using Cloud VPN or Cloud Interconnect. Now you can use on-premises hosts to send traffic from arbitrary internal IP addresses, including the non-RFC 1918 address space. Documentation
Cloud Load Balancing – multiple domain support for Google-managed SSL certificates: beta
Secure Cloud Load Balancing by default – without manually generating certificates – and create SSL certificates for multiple domains. Provide a custom domain name, update the DNS records, and a managed SSL certificate will be automatically provisioned and renewed. Documentation
