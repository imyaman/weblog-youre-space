GCP UPDATES | SEPTEMBER 3, 2019
AI & MACHINE LEARNING
Dialogflow – auto speech adaptation: beta
Enable this feature to improve your virtual agent’s speech recognition accuracy based on information included in your Dialogflow agent. Define product or service names for your company, allow agents to more effectively recognize numerical sequences, and more. Documentation
AI Platform Data Labeling Service: beta
Submit samples of your data to reviewers who annotate and return them in a format suitable for training machine learning models. New features of the service include an updated user interface and the ability to review human-labeled results and monitor model performance on AI Platform. Documentation
NETWORKING
Traffic Director – new routing rules and traffic policies: beta
Gain greater control of traffic within your Traffic Director deployment with new route rules and traffic policies. You can now enable features including traffic splitting, traffic steering, fault injection, circuit breaking, and mirroring. Documentation
Private Google Access for on-premises hosts: beta
Enable on-premises hosts to reach all Google APIs and services over a Cloud VPN or Cloud Interconnect connection from your data center to Google Cloud Platform without the need for external IP addresses. Documentation
API PLATFORM & ECOSYSTEMS
Cloud Healthcare API – now available in Tokyo and the Netherlands: beta
Access the Cloud Healthcare API in two additional regions: the Netherlands (europe-west4) and Tokyo (asia-northeast1). The API offers a managed solution for storing and accessing healthcare data and it acts as a bridge between existing systems and apps hosted on Google Cloud. Documentation
Cloud Healthcare API – Stackdriver error logging: beta
View all of your logs for errors triggered in DICOM import, FHIR import, and FHIR export requests in a unified Stackdriver dashboard. Documentation
COMPUTE
Kubernetes Engine – containerd: GA
Use containerd with the Kubernetes Container Runtime Interface to natively execute apps within containers and handle interactions with the underlying host operating system. Documentation
MIGRATION
Velostrata is now Migrate for Compute Engine: 4.5 GA
This new release comes with a new name and features to help you migrate virtual machines to Compute Engine from on-premises data centers, AWS, and Microsoft Azure, with added support for Windows 2019, Amazon Linux, and cloud-to-cloud migration of AWS instances. Documentation
SECURITY
Cloud Security Scanner: GA
Easily set up, schedule, manage, and run scans of your App Engine, Compute Engine, and Kubernetes Engine apps. Detect common vulnerabilities, including cross-site scripting (XSS), Flash injection, mixed content (HTTP in HTTPS), and outdated and insecure libraries. Product page
MANAGEMENT TOOLS
Billing – new billing account overview page: GA
Get an at-a-glance summary of your charges to date, estimated end-of-month charges, and any credit balances from within your Google Cloud Platform Console with the newly redesigned billing account overview page. Documentation | Blog
STORAGE & DATABASES
Cloud Spanner – now available in São Paulo: GA
Now you can access Cloud Spanner in a new region: São Paulo, Brazil, southamerica-east1. Documentation
NetApp – Cloud Volumes Service: beta
NetApp Cloud Volumes Service offers a fault-tolerant, scalable platform for creating cloud-native file systems. Migrate existing enterprise and industry-specific apps to GCP, or build new machine learning and Kubernetes-based apps that require persistent storage. Documentation
SECURITY
Cloud Identity-Aware Proxy – TCP forwarding: GA
Control access to GCP-hosted services and adopt an app-level identity control model instead of using network-level firewalls. This launch adds TCP capability to enable administration of cloud resources without requiring public IP addresses. Documentation
