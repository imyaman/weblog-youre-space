ICO 회사 설립하기

  • Posted on April 5, 2018
  • by Sangyong Gwak

ICO를 주사업으로 하는 회사를 만들려면 어떻게 해야할까요?

그 방법과 주의사항을 이야기 해보려고 합니다. 따라하기 문서도 만들어보고자 합니다.

회사가 어떤 방법으로 암호화폐를 한화(원)으로 바꾸었다면, 관련된 세금이 있을까?
부가세를 내야할까? 법인세를 내게 될까?

회사가 임직원에게 암호화폐를 급여성으로 제공해도 될까?
임직원이 어떠한 방법으로 암호화폐를 한화(원)으로 바꾸었다면, 관련된 세금이 있을까?
소득세를 내야할까?

임걱정이 범죄행위로 발생한 수익(예를 들어 은행에서 훔친 돈)을 회사에 투자하고, 임직원에게 암호화폐를 급여성으로 제공하고, 임직원이 암호화폐를 현금으로 바꾸고, 임직원이 임걱정에게 현금을 조금씩 제공한다면 문제가 없을까?
자금세탁에 악용될 수 있을까? 이걸 막으려면 어떻게 해야할까?

에스토니아 Estonia 에 회사를 설립하고 싶은 분 imyaman@netscape.net 으로 연락 주세요.
에스토니아에 회사 등록하는거 도와드려요.
다르게 표현하면, 에스토니아에 회사를 등록하고 ICO 하시려는 분들께 약간의 설립 비용이랑 수고비 받고 넘겨드리려고요.
private limited company(에스토니아식 표현으로 OÜ) 형태의 회사로 만들게 되고,
제가 등록하면서 회사의 이사(members of the management board)가 되긴 하지만, 적절한 시점에 그것도 변경 해드리려고요.

Perl, json, 한글 출력

  • Posted on April 5, 2018
  • by Sangyong Gwak

#!/usr/bin/env perl

use utf8;
use strict;
use Encode;
use JSON;

binmode(STDOUT, ":utf8");

my ($line, $content, $perl_hashref, $key, $json);

$content = qq/{"별명":"컴맹"}/;
$perl_hashref = from_json($content);
for $key (sort keys %{$perl_hashref}){
  print $key . " : " . $perl_hashref->{$key} . "\n";
}

my $json = to_json($perl_hashref, {utf8 => 0, pretty => 1});
print $json;

아래 문서의 설명을 보면,
http://search.cpan.org/~ishigaki/JSON-2.97001/lib/JSON.pm

to_json : Converts the given Perl data structure to a Unicode string by default.
from_json : The opposite of to_json: expects a Unicode string and tries to parse it, returning the resulting reference.

encode_json과 decode_json은 UTF-8 encoded, binary string을 입력 받거나 출력한다고 되어있다.
encode_json과 decode_json을 써서는 json에서 한글 출력이 번거로웠다.
그래서, binmode와 to_json, from_json을 사용해야겠다.
