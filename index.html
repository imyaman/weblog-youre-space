getaddrinfo EAI_AGAIN

  • Posted on February 16, 2020
  • by Sangyong Gwak

If you see these words "getaddrinfo", "EAI_AGAIN"

npm ERR! code EAI_AGAIN npm ERR! errno EAI_AGAIN npm ERR! request to https://registry.npmjs.org/@fnproject%2ffdk failed, reason: getaddrinfo EAI_AGAIN registry.npmjs.org registry.npmjs.org:443

You have to check /etc/hosts. "127.0.0.1 localhost" must be last, other "127.0.0.1" must be first.

Not good,

127.0.0.1 localhost 127.0.1.1 imyaman.youre.space imyaman

Good,

127.0.1.1 imyaman.youre.space imyaman 127.0.0.1 localhost
