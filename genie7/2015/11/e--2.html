e커머스 데이터 파헤치기-2편

​e커머스 매출 올리는 산수(算數) - 매출 올리는 방법 1

이 글은 중학생들도 이해할 수 있도록 쓰고 싶다는 생각을 했다. 집에 초등학교 5학년 딸이 있는데 아마도 이런 계산을 설명해주더라도 잘 이해를 못하거나 하고 싶어 하지 않을 것 같다. 그래서 초등학생이 이해하도록 하는 것은 과감히(?) 포기하고, 중학생 정도 되면 이해할 수 있지 않을까 하는 기대를 해본다.(못하면 할 수 없고!!!) 그래서 이 글이 너무 쉽게 쓰여져 가더라도 그냥 이해하고 가볍게 넘어가기를 바란다.
지난 e커머스 데이터 파헤치기-1편(e커머스 매출 올리는 산수(算數) - 매출액 계산하기) 에서 매출액을 계산하는 방법을 알아보았다. 어떤 사업에서든 매출을 계산하는 것은 매우 중요하다. 매출을 계산할 줄 알아야 매출을 높이는 방법도 찾을 수 있기 때문이다.
지난번 정리했던 매출을 올리는 방법 여러 가지 중에서 오늘은 방법 1에 대해 이야기해 보고자 한다.

방법 1을 찾아내기 위한 기본 공식은 다음과 같았다.

매출액1 = (상품1 X 판매수량1) + (상품2 X 판매수량2) + (상품3 X 판매수량3) + ... + (상품n X 판매수량n)

그리고 방법 1의 내용은 다음과 같다.

방법 1 : 상품1, 상품2, 상품3, ..., 상품n의 각각의 상품의 가격을 높인다.
이해하기 쉽게 예를 들어서 설명해 보자.

[사례 1]
인터넷 쇼핑몰 "문구닷컴"을 이용해서 자신이 보유한 상품을 판매하는 사업자는 Mr. Cho이다. Mr. Cho는 5가지의 상품을 가지고 있다. 5가지의 상품과 가격은 각각 다음과 같다.

- 검정 볼펜 : 1,000원
- 파란 볼펜 : 1,000원
- 빨간 볼펜 : 1,000원
-흰색 메모장 : 500원
- 노란색 메모장 : 500원

2015년 10월 7일 Mr. Cho는 인터넷 쇼핑몰 "문구닷컴"에서 다음과 같은 판매를 기록하였다.

- 검정 볼펜 : 100개
- 파란 볼펜 : 48개
- 빨간 볼펜 : 20개
- 흰색 메모장 : 20개
- 노란색 메모장 : 10개
[사례 1]에서의 매출액을 계산해 보자.

공식 "매출액 = (상품1 X 판매수량1) + (상품2 X 판매수량2) + (상품3 X 판매수량3) + ... + (상품n X 판매수량n)"에 상품과 판매수량을 각각 대입하여 계산해 보자. [사례 1]에서 상품의 수는 5가지 이므로 공식에서 n은 5가 된다. 그럼 대입하여 공식을 만들어 보자.

매출액 = (검정 볼펜 가격X 검정 볼펜 판매수량) + (파란볼펜 가격 X 파란 볼펜 판매수량) + (빨간 볼펜 가격 X 빨간 볼펜 판매수량) + (흰색 메모장 가격 X 흰색 메모장판매수량) + (노란색 메모장가격 X 노란색 메모장판매수량)
= (1,000원 X 100개) + (1,000원 X 48개) + (1,000원 X 20개) + (500원 X 20개) + (500원 X 10개)
= (100,000원) + (48,000원) + (20,000원) + (10,000원) + (5000원)
= 183,000원

2015년 10월 7일 매출액 : 183,000원
매출액을 계산해 보았다. 자 이제 매출액을 올리기 위해 방법 1을 적용해보자. 방법 1은 각각의 상품의 가격을 높이는 것이다. (여기서 잠깐! 공식의 적용을 위해 다른 외적인 요인은 발생하지 않는다고 가정하자. 예를 들면 가격을 올린 이후에 매출 수량이 변경되지 않는다는 것이다.)

매출액을 10% 증가시키고 싶어 진 Mr. Cho는 상품의 개별 가격을 10%씩 인상을 실시한다. 가격의 인상은 2015년 10월 8일 00시에 조정이 되었다고 가정하자.

그러면, 상품의 가격은 다음과 같이 변경된다.

- 검정 볼펜 : 1,100원
- 파란 볼펜 : 1,100원
- 빨간 볼펜 : 1,100원
-흰색 메모장 : 550원
- 노란색 메모장 : 550원

가격이 인상된 후 2015년 10월 8일Mr. Cho는 인터넷 쇼핑몰 "문구닷컴"에서 2015년 10월 7일과 같은 판매를 기록하였다. 이제 2015년 10월 8일의 매출을 공식에 의해 계산해보자.

매출액 = (1,100원 X 100개) + (1,100원 X 48개) + (1,100원 X 20개) + (550원 X 20개) + (550원 X 10개)
= (110,000원) + (52,800원) + (22,000원) + (11,000원) + (5500원)
= 201,300원

2015년 10월 8일 매출액 : 201,300원
2015년 10월 8일의 매출액은 2015년 10월 7일의 매출액보다 10%가 증가했다.

그러나!

여기서 우리는 외적 요인에 따른 변화가 발생하지 않는다고 가정했으나, 실제의 세계에서는 외적 요인이 반드시 발생할 수밖에 없고, 특히 동일한 상품의 가격의 변화는 판매 수량에 중요한 영향을 끼치게 된다. 오늘은 방법 1에 대한 매출액이 증가하는 방법을 확인하기 위해 특정 조건을 가정하고 계산을 하였다. 실제의 세계에서 방법 1을 활용하여 매출액을 증가시키기 위한 방법을 사용하다면, 다양한 실험을 통하여 상품의 가격을 어떻게 변경시키는 것이 좋을지 그 방법을 찾는 것이 중요하다.
