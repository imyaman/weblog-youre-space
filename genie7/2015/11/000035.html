e커머스 데이터 파헤치기-4편

  • Posted on November 9, 2015
  • by genie7

e커머스 매출을 올리는 수(算數) - 매출을 올리는 방법 3

주말이 되고 연휴가 있으면 평소보다 좀 더 많은 글을 쓰고, 생각할 시간을 갖게 될 것이라 예상해보지만, 연휴 끝나가는 시점엔 그 작은(?) 실천을 하지 못했음을 반성하게 된다. 전 직장에서 모시던 상무님은 해외 출장 중 비행시간이 가장 기다려지는 시간이라고 하셨다. 아무에게도 방해받지 않고 오랜 시간 책을 읽거나 잠을 자거나 할 수 있다는 것이 그 이유였다. 집에서 아무런 방해 없이 사는 나도 집중해서 글을 쓸 수 있는 시간이 출퇴근길 버스 안이라는 것은 참 아이러니하다. 시간은 오래 걸리지만 앉아서 출퇴근할 수 있는 버스 노선이 있는 덕분에 요즘은 자가용을 놓고 다니며, 버스 안에서 글을 쓰기 위해 집중해 본다.


본론으로 돌아가서 오늘은 커머스 매출을 올리기 위한 방법 3에 대해 이야기해 보려 한다. 매출액에 대한 기본 공식은 다음과 같다.
매출액1 = (상품1 X 판매수량1) + (상품2 X 판매수량2) + (상품3 X 판매수량3) + ... + (상품n X 판매수량n)

그리고, 매출액을 높이기 위한 방법 3은 다음과 같다.
방법 3 : n은 더 큰 수가 될 수 있도록 판매되는 상품을 늘린다.

n을 더 큰 수가 되도록 한다는 것은 판매하는 상품의 종류를 더 많이 구비하는 것을 의미한다. 팔수 있는 상품이 10종류에서 25종류로 늘어나면 추가된 15종류의 상품에서 추가로 판매가 발생할 수 있게 되는 것이다.

사례는 역시 앞의 글 에서 사용했던 사례를 그대로 사용해보자.

[사례 1]의 2015년 10월 7일의 상황에서 매출을 올리기 위한 방법 3을 적용하게 되면, 판매할 상품의 종류를 더 많이 구비해야 한다.어떻게 상품을증가시킬 수 있을지에 대해서는 별도로 이야기해 보도록 하고, 여기서는 역시나 산술적 계산만 해보도록 하자. 여기서 Mr. Cho는 다음의 5종류의 상품을 판매했었다.

- 검정 볼펜 : 1,000원
- 파란 볼펜 : 1,000원
- 빨간 볼펜 : 1,000원
- 흰색 메모장 : 500원
- 노란색 메모장 : 500원

이 5종류의 상품으로 발생한 2015년 10월 7일의 매출액은 183,000원이다.

2015년 10월 8일이 되었을 때, 각 상품의 가격은 아무것도 달라진 것이 없다. 그러나, 우리는 새로운 상품을 하나 더 판매할 계획이다. 어떤 상품을 구비할 것인지는 상품의 확대 전략에 따라 진행해야 하고, 그 전략에 대해서는 뒤에 이야기해 보도록 하자. 추가된 상품은 다음과 같다.

- 하늘색 메모장 : 1000원

그리고 이 메모장은 2015년 10월 8일 5개의 판매를 기록한다고 가정하고, 다른 기존의 상품은 2015년 10월 7일의 판매량과 동일하다고 가정하자. 즉,

- 검정 볼펜 : 100개
- 파란 볼펜 : 48개
- 빨간 볼펜 : 20개
- 흰색 메모장 : 20개
- 노란색 메모장 : 10개
- 하늘색 메모장 : 5개

그러면 매출액을 계산해보자.
매출액 = (검정 볼펜 가격X 검정 볼펜 판매수량) + (파란볼펜 가격 X 파란 볼펜 판매수량) + (빨간 볼펜 가격 X 빨간 볼펜 판매수량) + (흰색 메모장 가격 X 흰색 메모장판매수량) + (노란색 메모장가격 X 노란색 메모장판매수량) + (하늘색 메모장 X 하늘색 메모장 판매수량)
= (1,000원 X 100개) + (1,000원 X 48개) + (1,000원 X 20개) + (500원 X 20개) + (500원 X 10개) + (1000원 X 5개)
= (100,000원) + (48,000원) + (20,000원) + (10,000원) + (5000원) + (5000원)
= 188,000원

2015년 10월 8일 매출액 : 188,000원

새로운 상품 하늘색 메모장에 대한 매출 5,000원이 만큼 증가하였음을 확인할 수 있다.

이번 방법 3에서 우리는 새로운 상품을 판매한 것 이외에 기존 상품에 대한 아무 변화가 없었다. 새로운 상품을 추가하여 판매가 가능한 상품을 늘림으로 해서 매출이 증가하는 것을 확인할 수 있었다. 판매자인 Mr. Cho는 새로운 상품을 늘려서 매출을 증가시킬 수 있음을 알게 되었다. 그러나 실제 상황에서는 상품이 늘어났다고 해서 무조건 매출이 발생하는 것은 아니다. 그리고, 새로운 상품을 구성하는데 있어서도 매출이 발생하지 않을 상품을 늘이는 것보다도 매출이 발생할 상품을 늘이는 데 집중해야 할 것이다. 어떠한 상품을 추가하여 매출을 증가시킬지는 이야기할 다른 기회를 가져보도록 하자.
방법 1, 방법 2, 방법 3에서 우리는 다음의 방법을 통해 매출을 증가시키는 계산을 해보았다.
- 상품의 가격을 높이는 방법
- 판매 수량을 증가시키는 방법
- 판매할 새로운 상품을 추가하는 방법

이 방법은 "상품"이라는 관점에서 매출을 증가시키는 방법이다. "상품"의 가격, "상품"의 판매 수량, "상품"의 종류로 정리될 수 있다. 고객이 누구이거나 어떤 시장이냐에 대한 문제가 아닌 상품 자체의 속성을 통해 매출을 증가시키는 방법인 것이다.

다음에 정리할 내용은 "고객"이라는 관점에서 매출을 증가시키는 방법이다.
