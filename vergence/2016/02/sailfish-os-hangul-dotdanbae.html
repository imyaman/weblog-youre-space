Sailfish OS에서 한글을 쓰려면

Sailfish OS에서 한글을 쓰려면 돛단배 입력기를 설치하면 좋다.

  • https://openrepos.net/content/peremen/dotdanbae 에서 dotdanbae-0.3-1.noarch.rpm 파일을 다운로드하여 휴대폰에 저장한다.
  • 휴대폰에서 Terminal 명령을 실행한다.
  • devel-ssu 명령을 실행한다. 비밀번호를 입력해야하는데, 비밀번호는 설정의 Developer mode에서 정할 수 있다. 명령을 실행하고 나면 root 권한을 얻은 상태가 된다.
  • 돛단배는 jolla-xt9 패키지를 필요로한다. root 상태에서 pkgcon install jolla-xt9 를 입력하여 패키지를 설치한다.
  • root 상태에서 rpm -i dotdanbae-0.3-1.noarch.rpm 를 실행한다.
  • 설정의 Text input 메뉴에 가서 "Keyboards" 라는 버튼을 누른다. English와 한글(두벌식)을 선택한다.
  • 이후 글자를 입력할 때 키보드의 스페이스 키를 길게 누르면 한/영 전환이 된다.
