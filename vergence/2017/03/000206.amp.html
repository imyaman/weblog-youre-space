Firebase와 Vue.js

March 21, 2017 March 21, 2017 by Sangyong Gwak

Firebase bindings for Vue.js라고 설명되어있는 VueFire라는게 있습니다. 이것 없어도 v-for 같이 목록을 표시하려는 구문에 Firebase 쿼리 결과를 그대로 전달해주어도 템플릿에서 데이터에 편하게 접근할 수 있었습니다. 그래서 Firebase와 Vue.js에 낙점해야겠다고 생각했습니다, 어제 저녁 시점까지만 해도.

오늘 아침 생각은, Parse Server로 낙점, UI 프레임워크를 쓰지 말고 단순한 JavaScript로 해결해야겠다입니다. 뭐 대단하고 복잡한 걸 만들 거라고. 에휴.

Frontend의 rounting 처리해주는 라이브러리는 찾아서 넣고, jquery 정도만 써서 낼롬 만들어야지 싶습니다.
