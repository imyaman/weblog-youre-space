React Router v4

By Sangyong Gwak on March 22, 2017 8:30 AM

React Router로 삽질을 했습니다. 여러 경로를 웹 문서 하나에서 처리하기 위해 frontend router를 쓰는데, React Router는 그 중 하나입니다. 예를 들면, /home, /about, /contact 주소로 요청한 걸 모두
index.html이 처리, 그것도 frontend에서.
React Router 최신 버전은 v4.0.0인데, 별 생각 없이 그 버전를 선택했습니다. react-router-v4 ...
문서와 예제는 대부분 v3.x에 맞게 되어있는데, v4에서 많이 달라진 줄 모르고 그대로 쓰려고 한 거죠.

React Router를 쓴다면, 현재는 v3으로. 미래를 위해선 v4로.

일이 아니고 취미생활이라 참 다행입니다.
