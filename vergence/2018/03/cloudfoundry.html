cloudfoundry 써보세요

By Sangyong Gwak on March 9, 2018 2:49 PM

$ cf login -a https://api.sys.sec.mzdemo.net # 대화식으로 아이디, 비밀번호 입력

$ cd # https://github.com/imyaman/sourcey-buildpack 를 clone 하여 example/perl-dancer2 디렉토리로 이동

$ vi manifest.yml # name을 변경. perl-dancer2-imyaman-cheerful-squirrel.apps.sec.mzdemo.net 이런 주소의 앞부분에 쓰임 )

$ cf push

하시면 바로 웹 사이트가 하나 딱 나올거예요. ^^
이후에 코드 변경은 app.pl 등을 수정하시면 되고요.

cf-cli 설치는 여기 참고하세요~
https://github.com/cloudfoundry/cli
