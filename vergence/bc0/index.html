아래아 한글로 movabletype에 글 쓰기

아래아 한글 2010 SE 버전부터인 것 같은데, 도구 -> SNS 서비스 -> 블로그로 올리기 기능을 유용하다고 생각하고 있다.
movabletype 2.6을 사용하고 있다. 설정에서 API 주소를 movabletype이 설치된 곳의 mt-xmlrpc.cgi으로 입력하면 된다.
글을 블로그로 올리면 오류 메시지가 나오지만, 실제로는 정상적으로 등록이 된다.
